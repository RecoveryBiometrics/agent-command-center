# Spanish style rules (Latin American neutral)

Every Spanish article AND hub must follow these. Fact-checker / Opus reviewer rejects on violation.

## Dialect: Latin American neutral, NOT rioplatense

**Use "tú" + neutral verb forms:**
- `tienes`, `necesitas`, `puedes`, `diriges`, `quieres`, `eres`

**NEVER use voseo (Argentine/Uruguayan):**
- ❌ `tenés`, `necesitás`, `podés`, `dirigís`, `querés`, `sos`

**Why:** voseo signals Argentina/Uruguay specifically. Most LatAm readers (Mexico, Colombia, Chile, Peru) read it as foreign. Neutral "tú" reads natural everywhere.

## Translation discipline

NO literal translation from English. Spanish has its own rhythm. Translate intent, not words.

Bad (literal): `En este post vamos a explorar...`
Better: Just start with the content. Drop the framing entirely.

Bad (literal): `Vamos a sumergirnos en...`
Better: Start with the actual claim.

## Punctuation

- **No em-dashes anywhere** (Bill hates them, also AI tell)
- Use comma, period, or parenthesis instead
- Spanish quotes preferred: `«»` for primary, `""` for nested. Curly straight quotes acceptable.
- Spanish question/exclamation pairs: `¿...?` and `¡...!` (both opening AND closing required)

## Banned AI-tell phrases (strip every occurrence)

These are the Spanish equivalents of the English banlist in `voice.md`:

| Banned | Why | Use instead |
|---|---|---|
| `aprovechar` | direct translation of "leverage" — corporate-speak | `usar` |
| `utilizar` | overformal | `usar` |
| `sumergirnos` / `vamos a sumergirnos` | direct of "dive into" | start with the content |
| `en el mundo actual` | direct of "in today's world" | cut |
| `llevar al siguiente nivel` | direct of "next level" | specific outcome |
| `desbloquear` | direct of "unlock" | specific verb |
| `revolucionar` | hype | specific change |
| `ecosistema` (non-biological) | SaaS-speak | specific term |
| `journey` (in Spanish text) | English borrow, lazy | `proceso` or `recorrido` if appropriate |
| `robusto` | empty | specific capability |
| `holístico` (non-medical) | wellness-speak | specific |
| `onboarding` (untranslated) | English in Spanish text | `incorporación` OR describe what happens |
| `seamlessly` / `sin problemas` (filler) | marketing fluff | cut |
| `en conclusión` / `para concluir` | essay filler | just end |
| `Ya sea principiante o experto` | pandering | name the actual reader |

## Pricing references

If pricing is mentioned, use the canonical numbers from `businesses/globalhighlevel.yaml`:
- Starter: $97/mes
- Agency Pro: $297/mes (or $497 if applicable)
- India: ₹8,000/month (Starter), ₹24,700/month (Agency)

Only mention specific dollar/peso/rupee amounts if you can cite or contextualize them. Don't invent client savings figures.

## Country names + currency

Default LatAm targeting: México, Colombia, Argentina (in that order of GHL Spanish search volume).
- Use `$` for USD, `MXN $` or `COP $` for local pesos when relevant
- Don't claim specific amounts of cost-savings without article-supported sources

## Native research mandate

Per-market: when writing for LatAm Spanish, research must include LatAm-specific:
- Tools (e.g., Mercado Pago, not just Stripe)
- WhatsApp as primary channel (not SMS)
- Spanish-speaking competitors (Clientify, Holded, Treble.ai)

Don't translate-once from English research. The references are different.
