- Ops activity summary from stage 01

1. Read voice.md for tone and hard rules
2. Read the platform template file for post structure and examples
3. Write posts using real numbers from the ops summary
4. Each post must be a genuinely different angle and story
5. No repeated numbers or themes across posts
6. Read each post back — if it sounds like AI, rewrite

- Passed to stage 03 (post)
