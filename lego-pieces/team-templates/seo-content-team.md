# SEO Content Team
**Skill:** SEO Content Pipeline

## What This Team Does
Scrapes local events, writes unique articles tied to business niche, validates quality, deploys to production.

## Roles (in pipeline order)
1. **Researcher** → Scrapes real events from 3 platforms
2. **Fact Checker** → Validates events are future-dated
3. **Copywriter** → Writes articles with business niche tie-in
4. **Fact Checker** → Checks cross-city uniqueness (85% threshold)
5. **SEO Auditor** → Validates titles, meta, schema, auto-fixes
6. **Engineer** → Deploys to production via git push

## Deployed To
- **SafeBath Grab Bar** — GitHub Actions, daily 6am ET

## To Deploy to a New Business
Run `/deploy-team` and select "SEO Content Team"
