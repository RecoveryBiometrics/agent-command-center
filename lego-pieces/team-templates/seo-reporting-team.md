# SEO Reporting Team
**Skill:** SEO Content Pipeline

## What This Team Does
Fetches search performance data, checks indexing, analyzes wins/drops/gaps, mines for opportunities.

## Roles (in pipeline order)
1. **GSC Fetcher** → Pulls 28 days of Search Console data
2. **Indexing Inspector** → Checks which pages Google has indexed (200/run)
3. **SEO Analyst** → Writes attribution report (wins, drops, opportunities, gaps)
4. **Keyword Miner** → Finds search queries with no landing page

## Deployed To
- **SafeBath Grab Bar** — GitHub Actions, weekly Tuesdays 9am ET

## To Deploy to a New Business
Run `/deploy-team` and select "SEO Reporting Team"
