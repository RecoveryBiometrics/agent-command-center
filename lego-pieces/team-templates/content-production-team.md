# Content Production Team
**Skill:** Podcast Pipeline

## What This Team Does
Full podcast + blog pipeline: discovers content, generates audio, optimizes SEO, transcribes, publishes, writes blog, fact-checks.

## Roles (in pipeline order)
1. **Scheduler** → Orchestrates 25-hour cycles
2. **Content Scraper** → Finds content from website/YouTube/RSS/web search
3. **Audio Producer** → Generates two-host podcast via NotebookLM
4. **SEO Writer** → Claude writes optimized title, description, tags
5. **Transcriber** → Gemini converts audio to text with speaker labels
6. **Publisher** → Uploads to Transistor.fm → Spotify/Apple/Amazon
7. **Blog Researcher** → Searches web + Reddit for real data
8. **Blog Writer** → Claude writes 800-1200 word SEO blog post
9. **Blog Fact Checker** → Validates accuracy, catches fabricated stats
10. **Site Builder** → Generates static blog site

## Deployed To
- **GlobalHighLevel.com** — IONOS VPS, 24/7, 20 episodes/day

## To Deploy to a New Business
Run `/deploy-team` and select "Content Production Team"
