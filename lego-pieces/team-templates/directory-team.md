# Directory Team
**Skill:** SEO Content Pipeline

## What This Team Does
Discovers real local businesses, verifies they exist, scores quality, enriches with Google data.

## Roles (in pipeline order)
1. **Discoverer** → Finds real businesses via Gemini web search
2. **Verifier** → Confirms each business exists and is operating
3. **Quality Auditor** → Scores each listing 0-100
4. **Places Enricher** → Adds Google Places data (ratings, hours, photos)

## Deployed To
- **SafeBath Grab Bar** — GitHub Actions, daily 6am ET

## To Deploy to a New Business
Run `/deploy-team` and select "Directory Team"
