# Never fabricate claims about Bill (the user) or William Welch (the persona)

**This is absolute. No workarounds.**

Never write content, memos, bios, or marketing copy that claims:
- A number of clients served ("helped 50 plumbers", "500 agencies")
- A specific outcome ("grew revenue 3x for")
- A time-in-business claim ("10 years of experience")
- A named client or case study
- A quote attributed to Bill or William
- Expertise in a specific vertical Bill has not confirmed
- Any first-person anecdote ("I set up GHL for a dental office and...")
- Any authority signal ("Trusted by", "Used by")

**Why:** Bill runs the globalhighlevel.com affiliate site as William Welch. The business model depends on readers trusting the content. A fabricated stat that gets caught (by a reader, competitor, or Google E-E-A-T scoring) destroys the whole domain's credibility. It also violates his "Never make things up" rule already in memory.

**How to apply:**
- Content pipelines (5-blog, 6/7/9-blog, trades-blog, any new pipeline) MUST include a fact-check pass that strips any first-person claim, client count, revenue outcome, or unverifiable authority signal.
- When the copywriter agent wants to add social proof, it must use one of these honest alternatives:
  - Data-led framing: "The data shows X" (with a real source)
  - Category framing: "Agencies that run service businesses report Y" (only if a public source supports it)
  - Hypothetical framing clearly marked: "Imagine a two truck plumbing business..." (not "I worked with")
  - Silence: drop the social proof line entirely if none of the above apply
- When William Welch is named as author, the bio must stay factual: affiliate site owner writing about GHL. No fabricated credentials.
- If Bill later confirms specific claims (e.g., "yes I have set up GHL for 50 plumbers"), update this memory to list what is safe to say.

**Banned phrases for content (add to voice.md):**
- "I have worked with"
- "I have set up GHL for"
- "We have helped"
- "Trusted by hundreds of"
- "Join thousands of"
- "In my experience with X clients"
- Any specific count of customers, agencies, or businesses unless sourced externally
- Any dollar figure Bill has earned or saved a client
